Retail Sales Database Project

- Designed relational database schema
- Generated realistic data using Python (Faker)
- Built analytical SQL queries
- Created Views for reporting

Technologies:
SQL Server/
Python
-----------------------------------------------------
Description:
 Designed and implemented a relational database for a retail store. Generated realistic datasets using Python (Faker) for customers, products, orders, and payments. Wrote complex SQL queries with JOINs, aggregations, and GROUP BY to analyze sales and customer behavior. Created Views to simplify reporting.
 
تم تصميم وتنفيذ قاعدة بيانات علائقية لمتجر تجزئة. تم توليد بيانات شبه حقيقية باستخدام Python (Faker) للعملاء، المنتجات، الطلبات والمدفوعات. تم كتابة استعلامات SQL معقدة باستخدام JOIN وGROUP BY لتحليل المبيعات وسلوك العملاء. تم إنشاء Views لتسهيل التقارير.
